# 4.Raspbian Desktop

##<u>目的</u>
Raspbian の Desktop は最近まで軽量な LXDE が使われていたが、最近はおしゃれな PIXEL に変わった  
PIXCEL で用意されているアプリケーションを理解する

##<u>実習手順</u>
自身の gc15 に RemoteDesktop でログインする
